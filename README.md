## Hospital Drug Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/6628b0d535784b0a9e44074c0e3199eb/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
The system is divided into (roles): regular users, administrators.

**User Features:**
1. Login
2. Registration
3. Homepage
4. Personal Center (Change Password, Personal Information)
5. Announcement Information Management
6. Drug Information Management
7. Drug Pickup/Return Record Management

**Employee Features:**
1. Login
2. Registration
3. Homepage
4. Personal Center (Change Password, Personal Information)
5. Announcement Information Management
6. Drug Information Management
7. Inventory Management
8. Drug Pickup/Return Record Management
9. User Management

**Administrator Features:**
1. Login
2. Personal Center
3. Administrator Management
4. Basic Data Management (Drug Unit Management, Announcement Type Management, Drug Type Management)
5. Announcement Information Management
6. Drug Information Management
7. Drug Pickup/Return Record Management
8. Inventory Management (Inventory Management, Inventory Details Management)
9. User Management
10. Employee Management
